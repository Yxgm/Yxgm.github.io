clientHeight

获取元素的位置信息：

window.innerWidth 视口高度

document.body.clientHeight ：body的内容高度 （一般大于等于视口高度）

document.body.offsetHeight ：clientHeight - border

网页被卷去的高： document.body.scrollTop（常用）

当前节点距离视口顶部的高度 ：offsetTop （常用）

屏幕分辨率的高： window.screen.height;屏幕分辨率的宽： window.screen.width;

documentElement会返回一个文档对象的根结点（html标签）

getClientRects()获得节点信息拼接成node节点数组

getBoundingClientRect()获得单个节点位置信息

clientHeight：元素的高，但是不包括不可见区域（滚动的部分不计算）,标准盒的（content+padding） clientWidth同理。

offsetHeight: 相当于clientHeight+border的高度。标准盒（content+padding+border） 同时需要说明的是：⚠️margin的边距不进入计算

vue3中：获取屏幕的高度和宽度
  • 文档的 Element.clientWidth 是指一个文档使用 CSS pixels 单位表示的内部宽度，包括其 padding（不包括 border、margin 或垂直滚动条）。这就是视口的宽度。
  • Window.innerWidth 是用 CSS 像素单位表示的浏览器窗口的视口宽度，包括垂直滚动条。
  • Window.outerWidth 是指包括了浏览器外边框的窗口宽度。
