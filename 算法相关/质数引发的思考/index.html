质数引发的思考

乞丐版

升级版

为此，我们仍然需要更加复杂的数论推导。

对于合数 n，我们可以证明它一定有一个小于等于 n 的非平凡因数。这里的非平凡因数，指的是和1与他本身不同的因数。
如果不是，那么它所有的非平凡因数都是大于 n 的。我们任取其中一个和n不同的非平凡因数 m，那么存在整数 k 使 n=km，那么 k 也为 n 的非平凡因数，但是 k=nm<n ，矛盾。所以合数 n 一定有一个小于等于 n 的非平凡因数。

因此，我们只需要在2到 [n] 之间寻找 n 的因数。（这里的 [x] 表示不超过 x 的最大整数。）
